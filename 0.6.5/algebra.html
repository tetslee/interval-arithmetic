(static) pow(x, power) → {Interval}

Source:
  • lib/operations/algebra.js, line 124

Computes x^power given that power is an integer

If power is an Interval it must be a singletonInterval i.e. x^x is not supported yet

If power is a rational number use arithmetic.nthRoot instead

Examples
// 2^{-2}
Interval.pow(
  Interval(2, 2),
  -2
)  // Interval(1/4, 1/4)
// [2,3]^2
Interval.pow(
  Interval(2, 3),
  2
)  // Interval(4, 9)
// [2,3]^0
Interval.pow(
  Interval(2, 3),
  0
)  // Interval(1, 1)
// with a singleton interval
Interval.pow(
  Interval(2, 3),
  Interval(2)
)  // Interval(4, 9)
