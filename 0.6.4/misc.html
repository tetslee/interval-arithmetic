(static) difference(x, y) → {Interval}

Source:
  • lib/operations/misc.js, line 202

Computes the difference between x and y, i.e. an interval with all the values of x that are not in y

Examples
Interval.difference(
  Interval(3, 5),
  Interval(4, 6)
)  // Interval(3, prev(4))
Interval.difference(
  Interval(0, 3),
  Interval(0, 1)
)  // Interval(next(1), 3)
Interval.difference(
  Interval(0, 1),
  Interval.WHOLE
)  // Interval.EMPTY
Interval.difference(
  Interval(-Infinity, 0),
  Interval.WHOLE
)  // Interval.EMPTY
